# HappySappy

A cheerful Android application designed to bring happiness and positivity to users' daily lives.

## Features

- Mood tracking and journaling
- Daily affirmations and inspirational quotes
- Gratitude journal
- Habit tracking with positive reinforcement
- Meditation and mindfulness guidance
- Community support and sharing

## Getting Started

### Prerequisites

- Android Studio Arctic Fox or later
- Android SDK API level 30 or higher
- Java Development Kit (JDK) 11 or higher

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/HappySappy.git
   ```

2. Open the project in Android Studio

3. Sync the project with Gradle files

4. Build and run the application on an emulator or physical device

## Project Structure

```
HappySappy/
├── app/                    # Main application module
│   ├── src/
│   │   ├── main/          # Main source code
│   │   ├── debug/         # Debug-specific code
│   │   └── test/          # Unit tests
│   └── build.gradle.kts   # Module-level build script
├── gradle/                # Gradle wrapper files
├── build.gradle.kts       # Project-level build script
└── settings.gradle.kts    # Project settings
```

## Technologies Used

- Kotlin: Primary programming language
- Android Jetpack: 
  - ViewModel
  - LiveData
  - Room
  - Navigation Component
  - WorkManager
- Coroutines: Asynchronous programming
- Koin: Dependency injection
- Retrofit: Network requests
- Gson: JSON parsing
- Material Design: UI components
